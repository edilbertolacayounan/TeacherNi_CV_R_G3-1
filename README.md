# Clase práctica: Control de Versiones con Git y GitHub

En esta actividad aprenderás a usar Git y GitHub para trabajar con repositorios y realizar cambios colaborativos.

## Pasos a seguir:

1. **Clonar el repositorio del maestro**  
   - Copia la URL del repositorio en GitHub.  
   - En tu computadora, abre la terminal y escribe:  
     ```bash
     git clone <URL-del-repositorio>
     ```

2. **Crear un archivo de Python**  
   - Dentro de la carpeta del repositorio, crea un archivo llamado:
   -  si te llamas Juan Perez sera JP  
     `claseControlVersiones_(TUS INICIALES).py`

3. **Modificar el archivo de Python**  
   - Abre el archivo y agrega la siguiente línea:  
     ```python
     print("calculadora de [Nombre Apellido]")
     ```
   - Sustituye `[Nombre Apellido]` por tu nombre real.

4. **Subir el cambio al repositorio**  
   - Guarda el archivo y en la terminal escribe:  
     ```bash
     git add claseControlVersiones.py
     git commit -m "Agrego archivo con mensaje personalizado"
     git push
     ```

5. **Modificar el archivo README.md**  
   - Abre el archivo `README.md`.  
   - Agrega tu nombre y la hora en que subiste el archivo.  
     Ejemplo:  
     ```
     Estudiante: Juan Pérez - Hora: 10:35 AM
     ```

6. **Subir el cambio del README.md**  
   - En la terminal escribe:  
     ```bash
     git add README.md
     git commit -m "Agrego nombre y hora al README"
     git push
     ```
